You should be able to setup Cocoapods with any existing project. Here's how:

  • OPTIONAL: Create a new project if you don't already have a project created
  • cd into the project location
  • touch Podfile at the root of the project location
  • Edit the Podfile to include the dependencies that your project has
  • `pod install`
  • open .xcworkspace from now on.

    • A sample Podfile looks like this:
    [sourcecode language="bash"]
    platform :ios, '6.1'

    pod 'AFNetworking', '~>1.3.1'
    pod 'AFOAuth1Client'
    [/sourcecode]

    Hope this helps.
