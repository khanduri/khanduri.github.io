I hope to build on this blog with 2 goals in mind:

  1. I want to document what I learn (about the different technologies i play with)
  2. Share the discoveries so others hopefully find the info helpful (and help make it better)

A few technologies I wish to cover are:

  • Django (first project, a more detailed understanding of the framework)
  • Tastypie: The webservice API framework for Django
  • Getting a sample AngularJs app connected to Tastypie's REST endpoint (ORM data sources)
  • Haystack (on Solr): Modular search for Django
  • AngularJs and Tastypie (through haystack: Non ORM data sources)
  • Django-social-auth: Social authentication/registration for Django

Please keep in mind that I'm new to Django, Python and most of the technology I've listed above, myself therefore some of my methods \ code could be improved upon. If you find something that you disagree with or have suggestions for improvements, please comment back (or feel free to contact me).

I hope you all find the information helpful.
