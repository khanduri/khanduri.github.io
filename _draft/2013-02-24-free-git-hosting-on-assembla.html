AssemblaÂ is offering free private git repositories. Github is a good option too, but to use their free version your repo has to be public (private ones costs near $7/month).

Once you sign up follow the instructions to setup your environment for the first time. Assembla should provide with these instruction as the default landing page once you signup.

Once the setup in configured for the first time, to add more future projects in you assembla account, all you would need to do on your machine is:

git remote add origin git@git.assembla.com:<your_git_repo_name>.git
git remote -v # should verify that it was added in
git push origin master
