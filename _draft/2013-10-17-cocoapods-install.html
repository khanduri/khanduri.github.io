Cocoapods is an ObjectiveC package manager. If you're not already using it, you should give it a try.

Cocoapods can be easily installed and updated as instructed on their installation page. The command to install and upgrade cocoapods is:

[sourcecode language="bash"]
$ [sudo] gem install cocoapods
[/sourcecode]

Issues I ran into on OS X (Mountain Lion):

Please note you'll need the latest command line tools for xcode installed as well. You can manually download a dmg from Apple's Developer download site.

Another alternative is by opening up XCode -> Preferences -> Downloads -> Components and hitting the install next to Command Line Tools.

Skipping this step caused me to get the following (and was resolved once I had make installed through the command line tools):

[sourcecode language="bash"]
$gem install cocoapods
Building native extensions. This could take a while...
/usr/local/Cellar/ruby/2.0.0-p195/lib/ruby/2.0.0/rubygems/ext/builder.rb:47: warning: Insecure world writable dir /usr/local/bin in PATH, mode 040777
ERROR: Error installing cocoapods:
ERROR: Failed to build gem native extension.

/usr/local/Cellar/ruby/2.0.0-p195/bin/ruby extconf.rb
creating Makefile

make
sh: make: command not found
[/sourcecode]

I also had to link the installed pod into /usr/bin/

[sourcecode language="bash"]
$gem env # look for EXECUTABLE DIRECTORY
export PATH="EXECUTABLE DIRECTORY:$PATH" # gem bins
export PATH="/usr/local/Cellar/ruby/2.0.0-p195/bin:$PATH" # gem bins
[/sourcecode]
